# UnitaryPhaseLogic

命題を 1×1 ユニタリ行列（単位円上の複素数 `e^(iθ)`）として表現する論理フレームワークです。
論理演算はユニタリ変換として定義され、古典的な二値論理をより豊かな位相ベース構造へ拡張します。

## ライブデモ

🌐 **[インタラクティブな可視化を見る](https://worthmine.github.io/UnitaryPhaseLogic/)**

インタラクティブなデモでは、複素平面上でユニタリ位相論理の演算を視覚的に探索できます。

## 値体系

| ラベル | 行列値 | 位相 | cos(θ) | 古典写像 |
|-------|-------------|-------|--------|-------------------|
| T  (真)       |  1 | θ = 0    | > 0 | T |
| F  (偽)      |  i | θ = π/2  | = 0 | F |
| ¬T (反真)  | -1 | θ = π    | < 0 | —（古典領域外） |
| ¬F (反偽) | -i | θ = 3π/2 | = 0 | F |
| N  (Neither)    | その他 | その他の θ | — | — |

古典写像は `cos(θ)` を使います: 正 → T、0 → F、負 → 古典領域外。

## 演算子

| 演算子 | 定義 | 説明 |
|----------|-----------|-------------|
| NOT   | ¬A = i · A†      | 共役転置（位相反転）の後に 90° 回転 |
| AND   | A ∧ B = A · B    | 行列（複素）積 — 単位円上での位相加算 |
| OR    | A ∨ B = −i·(A·B) | AND の後に −90° 回転 |
| IMPLIES | A ⇒ B = B · A† | A から B への位相差 |
| EQAL  | A ⇔ B = (A⇒B) ∧ (B⇒A) | 相互含意 |

## 論理的帰結

### 普遍的に成り立つ定理（全ての位相）

以下の恒等式は、任意のユニタリ位相で表される命題 A, B（標準4値に限らない）に対して証明されます。8つすべてがテストで検証済みです。

| # | 定理 | 恒等式 | 備考 |
|---|---------|----------|------|
| 1 | **二重否定除去** | ¬(¬A) = A | 全ての位相 θ で成立 |
| 2 | **ド・モルガン則** | ¬(A ∨ B) = ¬A ∧ ¬B  および  ¬(A ∧ B) = ¬A ∨ ¬B | 完全な双対性 |
| 3 | **モーダス・ポネンス** | A ∧ (A ⇒ B) = B | 厳密に保存 |
| 4 | **排中律** | A ∨ ¬A = T | すべての位相で T に収束 |
| 5 | **矛盾律** | A ∧ ¬A = F | 常に定数行列 i を返す |
| 6 | **爆発律抑止** | ⊥ ⇒ B = −i · B | 矛盾は任意真理ではなく決定論的な −90° 位相シフトを与える |
| 7 | **対偶律** | A ⇒ B = ¬B ⇒ ¬A | 全ての位相で成立 |
| 8 | **背理法** | ¬A ⇒ ⊥ = A | 対偶律から直接導出 |

**証明**（1×1 ユニタリ行列 A = e^(iθ)）:

- *二重否定*: ¬(¬A) = i·(i·A†)† = i·(−i·A) = A ✓
- *排中律*: A ∨ ¬A = −i·(A·i·A†) = −i·i·(A·A†) = 1 = T ✓
- *矛盾律*: A ∧ ¬A = A·(i·A†) = i·(A·A†) = i = F ✓
- *モーダス・ポネンス*: A ∧ (A⇒B) = A·(B·A†) = B·(A·A†) = B ✓
- *対偶律*: A⇒B = B·A†; ¬B⇒¬A = (¬A)·(¬B)† = (i·A†)·(i·B†)† = (i·A†)·(−i·B) = (i)(−i)·A†·B = B·A† = A⇒B ✓
- *背理法*: 対偶律に B=⊥=F を代入: (¬A⇒⊥) = (¬⊥⇒A) = (¬F⇒A) = (T⇒A) = A·T† = A ✓  
  行列直接証明: ⊥·(¬A)† = i·(i·A†)† = i·(−i·A) = A ✓

### 追加の構造的帰結

| 帰結 | 恒等式 | 意味 |
|-------------|----------|---------|
| **含意の反射律** | A ⇒ A = T | すべての命題は自分自身を含意する |
| **ユニタリ恒等式** | A · A† = T | 命題とその共役の合成は常に True に潰れる |
| **普遍的同値** | A ⇔ B = T（任意の A, B） | 任意の命題対が同値（位相打ち消し） |

**普遍的同値の証明**:
A ⇔ B = (B·A†)·(A·B†) = B·(A†·A)·B† = B·T·B† = B·B† = T ✓

これは古典論理からの本質的な逸脱です: `EQAL` 演算子はオペランドに関わらず常に T を返します。これは値の一致ではなく、ユニタリ枠組みにおける構造的同値性を測ります。

### 代数構造

4つの標準値 {T, F, ¬T, ¬F} = {1, i, −1, −i} は、AND（複素乗算）に関して **巡回群 Z₄** を成します。NOT は共役と 90° 回転の合成です。

```
NOT cycle:  T → F → T   (位数 2)
            ¬T → ¬F → ¬T   (位数 2)
AND cycle:  T → F → ¬T → ¬F → T   (位数 4)
```

N（Neither）— {0, π/2, π, 3π/2} 以外の位相 — はすべての二項演算で **吸収的** です: どちらかのオペランドが N なら結果は N です。

### 古典論理との相違

以下は、古典写像 `cos(θ)` を適用したときに古典二値論理と異なる結果になるケースです。

| 演算 | UPL 結果 | 古典期待値 | 理由 |
|-----------|-----------|-----------------------|--------|
| F ∧ F     | ¬T        | F | 位相加算: π/2 + π/2 = π |
| T ∨ T     | ¬F        | T | AND 後に −90° 回転: 0 − π/2 = −π/2 |
| F ⇒ T     | ¬F        | T | UPL では空虚真が成立しない |

これらの差異は設計上のものです。UPL は二値真理への射影ではなく正確な位相を追跡するため、古典写像ではこれらをエラーではなく警告として扱います。

## 使い方

```python
import numpy as np
from UnitaryPhaseLogic import UnitaryPhaseLogic

T = UnitaryPhaseLogic(0.0)           # True  (θ = 0)
F = UnitaryPhaseLogic(np.pi / 2)     # False (θ = π/2)

print(T.NOT())          # UnitaryLogic[θ=90.0°]  → F
print(F.NOT())          # UnitaryLogic[θ=0.0°]   → T
print(T.AND(F))         # UnitaryLogic[θ=90.0°]  → F
print(T.OR(F))          # UnitaryLogic[θ=0.0°]   → T

A = UnitaryPhaseLogic(np.radians(45))
print(A.OR(A.NOT()))    # UnitaryLogic[θ=0.0°]   → T  (排中律)
print(A.AND(A.NOT()))   # UnitaryLogic[θ=90.0°]  → F  (矛盾律)
```
