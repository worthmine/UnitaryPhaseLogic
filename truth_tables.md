# UnitaryPhaseLogic — 真理値表

角度表記は弧度法。古典論理への写像は cos(θ) で行う (cos(θ) < 0 の点は考察外):

| ラベル | 行列値 | 位相 | cos(θ) | 古典論理 |
| --- | --- | --- | --- | --- |
| T  (真)      |  1 | θ = 0    | > 0 | T |
| F  (偽)      |  i | θ = π/2  | = 0 | F |
| ¬T (反真)    | -1 | θ = π    | < 0 | — |
| ¬F (反偽)    | -i | θ = 3π/2 | = 0 | F |
| N  (Neither) | その他 | θ = その他 | — | — |
